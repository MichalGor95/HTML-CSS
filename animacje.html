Rozdział CSS: Animacje CSS

Animacje w CSS pozwalają tworzyć ruch, efekty wizualne i płynne zmiany bez JavaScript. To ważny dział przy nowoczesnych stronach i często pojawia się na INF03.

1. Czym są animacje?

Animacja CSS zmienia wygląd elementu w czasie.

Można animować:

  • pozycję
  • kolor
  • obrót
  • rozmiar
  • przezroczystość
  • cień

Dzięki temu przycisk może się poruszać, karta może się pojawiać, a element może obracać się lub pulsować.

2. @keyframes - definicja animacji

Najpierw tworzymy przebieg animacji. 

@keyframes przesun{

    from{
        transform: translateX(0);
    }

    to{
        transform: translateX(200px);
    }

}

Można też użyć procentów: 

@keyframes kolory{

    0%{
        background:red;
    }

    50%{
        background:orange;
    }

    100%{
        background:green;
    }

}

3. animation-name

Wskazuje nazwę animacji przypisanej do elementu. 

.box{

    animation-name: przesun;

}

4. animation-duration

Czas trwania animacji. 

.box{

    animation-duration: 2s;

}
Wartość Znaczenie
1s jedna sekunda
300ms 0.3 sekundy
5s pięć sekund

5. animation-delay

Opóźnia rozpoczęcie. 

.box{

    animation-delay: 2s;

}

Element poczeka 2 sekundy i dopiero ruszy.

6. animation-iteration-count

Ile razy ma się wykonać. 

.box{

    animation-iteration-count: 4;

}

Bez końca:

.box{

    animation-iteration-count: infinite;

}

7. animation-direction

Kierunek animacji.

Wartość Działanie
normal standardowo
reverse od końca
alternate tam i z powrotem 
.box{

    animation-direction: alternate;

}

8. animation-timing-function

Tempo ruchu.

Wartość Efekt
linear stała prędkość
ease płynny start i koniec
ease-in wolny start
ease-out wolny koniec 
.box{

    animation-timing-function: ease-in;

}

9. animation-fill-mode

Co ma zostać po zakończeniu animacji.

Wartość Znaczenie
none wraca do początku
forwards zostaje na końcu
backwards ustawia stan początkowy wcześniej

10. Skrócona składnia animation

Zamiast wpisywać każdą właściwość osobno, można użyć jednej właściwości animation. 

.box{

    animation: ruch 2s linear 1s infinite alternate;

}

Kolejność wartości:

Wartość Znaczenie
ruch nazwa animacji
2s czas trwania
linear tempo animacji
1s opóźnienie startu
infinite powtarzanie bez końca
alternate tam i z powrotem

Ten zapis działa tak samo jak: 

.box{

    animation-name: ruch;
    animation-duration: 2s;
    animation-timing-function: linear;
    animation-delay: 1s;
    animation-iteration-count: infinite;
    animation-direction: alternate;

}

Skrócona składnia jest wygodniejsza, gdy ustawiasz kilka parametrów naraz.
